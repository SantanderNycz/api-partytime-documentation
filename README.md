# PartyTime

PartyTime é uma aplicação para gerenciamento de festas e eventos, integrando funcionalidades de cadastro, listagem e organização de eventos de forma prática e intuitiva.

## Repositórios

- **Front-end:** [API-PartyTime-Front](https://github.com/SantanderNycz/api-partytime-front)  
- **Back-end:** [API-PartyTime-Back](https://github.com/SantanderNycz/api-partytime-back)  

## Funcionalidades principais

- Cadastro e login de usuários  
- Criação, edição e exclusão de eventos  
- Consulta de eventos por usuário ou categoria  

## Tecnologias

- **Front-end:** React, HTML, CSS, JS  
- **Back-end:** Node.js, Express, MongoDB  

## Como rodar

1. Clone os repositórios front e back.  
2. Instale as dependências com `npm install`.  
3. Configure variáveis de ambiente conforme necessário.  
4. Rode `npm start` no back e no front.
